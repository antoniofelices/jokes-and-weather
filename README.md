## Documentation

### Installation

1. Clone this repo

```bash
$ git clone https://github.com/antoniofelices/filtering-sorting-movies .
```

2. Display branchs

```bash
$ git branch --all
```

3. Checkout branch vitest

```bash
$ git checkout vitest
```

4. Install packages

```bash
$ npm install
```

5. Run tests

```bash
$ npm test:watch
```

6. Run coverage

```bash
$ npm coverage
```
